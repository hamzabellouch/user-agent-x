## Reporting a Vulnerability

If you discover a security vulnerability, privacy issue, or any behavior that could negatively impact users or **User-Agent-X Beta**, please report it responsibly.

### Before Reporting

Please ensure that:

- The issue is reproducible.
- You are using the latest supported version of **User-Agent-X Beta**.
- The issue is not caused by modified builds, unsupported browsers, or third-party extensions.

### How to Report

You can report vulnerabilities through:

- GitHub Issues (for non-sensitive reports)
- Private communication channels (for sensitive vulnerabilities or exploit details)

When submitting a report, please include:

- Browser name and version
- Operating system and version
- User-Agent-X Beta version
- Steps to reproduce the issue
- Screenshots, logs, or console errors (if available)
- A clear description of the potential impact

### Response Policy

Security reports are reviewed as quickly as possible.

If a report is confirmed:

- The vulnerability will be investigated and resolved.
- A fix may be be included in the next release.
- Credit may be given to the reporter upon request.

If a report cannot be reproduced, lacks sufficient information, or is determined not to be a security issue, it may be closed without further action.

---

## Security Notes

**User-Agent-X Beta** runs locally within your browser and allows users to view, customize, or spoof the browser's User-Agent string for testing and compatibility purposes. Depending on its features and configuration, it may interact with browser APIs and modify request headers.

To help maintain a secure experience, users are encouraged to:

- Keep **User-Agent-X Beta** updated to the latest version.
- Install the extension only from official sources.
- Review the permissions requested by the extension before installation.
- Avoid installing modified or unofficial builds.
- Use User-Agent spoofing responsibly and only for legitimate testing or compatibility purposes.
- Report any unexpected behavior that could affect security or privacy.

Security and privacy are core priorities of the project, and improvements will continue to be incorporated as **User-Agent-X Beta** evolves.
